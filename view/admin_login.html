Calendario sobre el cual se listas todos los cobros y pagos del mes, ya sean desde las propias facturas, o mediante el plugin de tesoreria.
Permite añadir pagos y cobros manualmente.

{if="in_array('agenda_de_prevision_de_entrada_de_mercancia',$GLOBALS['plugins'])"}   Agenda previsión de entrada de mercancía {else}   Agenda previsión de entrada de mercancía {/if}

Calendario que permite ver la previsión de entradas de artículos en los pedidos de compra.
Además de incorporar un buscador de artículos basado en la fecha prevista de pedidos pendientes.

{if="in_array('extraccion_qlik',$GLOBALS['plugins'])"}   Extracción Qlik {else}   Extracción Qlik {/if}

Plugin para permitir extraer datos de documentos de compra o venta en XLS para ser utilizados en herramientas como Qlik o cualquier herramienta externa para analizar datos.
Relaciona datos entre las tablas referenciadas entre si en las propias de los documentos (por ejemplo: clientes, artículos, famílias, grupos de clientes, ...)

{if="in_array('firmar_documentos',$GLOBALS['plugins'])"}   Firmar documentos {else}   Firmar documentos {/if}

Permite recoger firmas desde una pantalla táctil (smartphone, tablets o similares).
Soportado desde presupuestos, pedidos, servicios, albaranes y facturas de venta.

{if="in_array('plantillas_html2pdf',$GLOBALS['plugins'])"}   Plantillas HTML2PDF {else}   Plantillas HTML2PDF {/if}

Permite generar plantillas de impresión a partir de plantillas preparadas en código HTML, incluyendo la impresión de firmas.
Incluye soporte para realizar plantillas propias en otra carpeta, para no perder los cambios entre actualizaciones.

{if="in_array('expedientes',$GLOBALS['plugins'])"}   Expedientes {else}   Expedientes {/if}

Plugin para Proyectos, ideal para agrupar documentos de compra y venta en expedientes y documentos adjuntos relaciones con dicho expediente, proyecto, obra etc.
Es ideal para empresas que realizan proyectos de cualquier tipo, Obras, o Trabajos, ya que permite agrupar todo lo relacionado con un proyecto.
